---

## 🚀 Présentation du Projet

Ce portfolio est bien plus qu'une simple page statique. Il a été conçu pour refléter la rigueur technique et le sens de l'architecture d'un développeur backend, tout en s'appuyant sur des visuels "premium" et des interactions dynamiques.

### Technologies Utilisées
- **Fondations** : HTML5 sémantique, CSS3 (variables natives, Grid & Flexbox).
- **Animations** : [GSAP](https://greensock.com/gsap/) (ScrollTrigger) pour des transitions fluides au défilement.
- **Visuels 3D** : [Three.js](https://threejs.org/) pour l'arrière-plan interactif du Hero.
- **Expérience Utilisateur** : Implémentation manuelle d'un "Smooth Scroll" (LERP) pour une navigation feutrée.
- **Multilingue** : Système i18n sur mesure supportant le Français, l'Anglais et le Japonais.

---

## ✨ Fonctionnalités Clés

- **Système de Grille Bento** : Présentation moderne des projets avec un effet "tilt" interactif au survol de la souris.
- **Horloge Dynamique** : Une horloge synchronisée en temps réel sur le fuseau horaire du Bénin (UTC+1), quel que soit le lieu de consultation.
- **Curseur Personnalisé** : Un curseur intelligent composé d'un point focal et d'un anneau de poursuite réactif aux éléments interactifs.
- **Protection Anti-Scraping** : L'adresse e-mail est générée dynamiquement via JavaScript pour limiter le spam par les robots.
- **Optimisation SEO** : Structure HTML5 optimisée pour le référencement et Meta tags descriptifs.

## 🛠️ Architecture du Projet

Le projet suit une structure simple et modulaire pour faciliter la maintenance :

```text
.
├── index.html    # Structure principale et contenu
├── style.css     # Design système et styles globaux
├── main.js       # Logique interactive, animations et i18n
└── README.md     # Documentation du projet
```

## 💻 Développement Local

Pour lancer ce projet sur votre machine en vue de modifications :

1.  Clonez le dépôt :
    ```bash
    git clone https://github.com/egbami/BPW.git
    ```
2.  Accédez au dossier :
    ```bash
    cd BPW
    ```
3.  Lancez le fichier `index.html` via une extension comme **Live Server** (VS Code) ou ouvrez-le directement dans votre navigateur préféré.

---

## ☁️ Déploiement sur Vercel

Le projet est optimisé pour un déploiement rapide sur **Vercel** :

1.  Connectez votre compte GitHub à Vercel.
2.  Importez le dépôt `bpw`.
3.  Vercel détectera automatiquement qu'il s'agit d'un projet "Plain HTML" (aucun build step requis).
4.  Cliquez sur **Deploy**. Votre portfolio sera en ligne en moins de 30 secondes !
